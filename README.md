# git-commands


## Commands

### git-mvh

> Git command to rename file in repository without loosing file history

```bash
sudo wget https://raw.githubusercontent.com/digaverse/git-commands/master/src/bash/git-mvh.sh -O /bin/git-mvh && sudo chmod +x /bin/git-mvh
```
**usage**

```
git mvh <src> <dest>
```
